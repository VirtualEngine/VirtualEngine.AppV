Custom Virtual Engine App-V 5.0 PowerShell modules (v1.0)

INSTALLATION:

Per User
  Extract the .ZIP file into the %Documents%\WindowsPowerShell\Modules\ folder

Per Machine
  Extract the .ZIP file into the %WINDIR%\system32\WindowsPowerShell\V1.0\Modules\ folder

IMPORTING MODULES

Run the following command
  Import-Module VirtualEngine-AppV5

VALIDATE INSTALL

Run the following command:
  Get-Command -Module VirtualEngine*

DOCUMENTATION

Run the following (as an example):
  Get-Help Save-AppV5FileReport -Full

or check out http://virtualengine.co.uk/downloads/appv5/
